# Healthcare Claims & Patient Risk Analysis

## Overview
This project analyzes healthcare claims and patient utilization data to identify trends in healthcare costs, readmission risk, and patient behavior. The analysis was performed using SQL, Python, Tableau, and Excel.

The goal of the project was to uncover actionable insights related to:
- Healthcare utilization
- Claims cost drivers
- Readmission patterns
- Patient demographics
- Department-level spending

---

## Tools & Technologies
- SQL
- Python (Pandas, NumPy, Matplotlib)
- Tableau
- Excel

---

## Key Analyses
### Claims Cost Analysis
- Identified medical conditions associated with highest healthcare costs
- Analyzed departmental spending trends

### Readmission Analysis
- Compared claim costs between readmitted and non-readmitted patients
- Identified demographic trends associated with readmission risk

### Patient Utilization
- Evaluated healthcare utilization across age groups and admission types
- Analyzed hospital visit frequency and length of stay

---

## Dashboard
The Tableau dashboard includes:
- KPI summary cards
- Claims cost trends
- Readmission analysis
- Department utilization
- Patient demographic insights

---

## Key Insights
- Older age groups showed higher average healthcare claim costs
- Emergency admissions were associated with increased spending
- Certain chronic conditions contributed disproportionately to total claims cost
- Readmitted patients demonstrated higher average healthcare utilization
